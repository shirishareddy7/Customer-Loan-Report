# Customer-Loan-Report (Interactive Dashboard creation using Power BI)

## 🎯 Project Objective
To analyze loan data and build an interactive Power BI dashboard to understand lending patterns, customer segments, and risk factors.

##  Dataset Used
- <a href="https://github.com/shirishareddy7/Customer-Loan-Report/blob/main/Power%20BI%20Project%20Bank%20Loan.pbix">Dataset</a>

## 📊 Questions (KPIs)
- Year-wise total loan amount trend  
- State-wise and month-wise loan status distribution  
- Loan distribution by home ownership type  
- Verified vs non-verified customer payment comparison  
- Revolving balance distribution by grade and sub-grade  

## ⚙️ Project Process
- Imported and cleaned dataset using Power Query  
- Transformed data and created data model  
- Built DAX measures for KPI calculations  
- Designed interactive dashboard with slicers and filters  

## 📌 Key Insights
- Loan amount increased significantly from 2007 to 2011  
- Most customers belong to Rent and Mortgage categories  
- Verified customers show higher repayment contribution  
- Loan activity is higher in later months of the year  
- Loan grades show variation in risk and balance distribution  


## ✅ Final Conclusion
The dashboard highlights key trends in loan distribution and repayment behavior. 
It helps in identifying risk patterns and supports data-driven decision-making in lending.
